### Nodejs + PostgreSQL + Docker

### Run auto reload development

```js
npm start
```

### Run docker compose

```js
docker compose up
```

### Run watch ts -> js

```js
npm run build
```

### Run production

```
npm run build
```

```js
npm run dev
```

### Auto compile scss -> css

```js
npm run watch
```

### Format all code

```js
npm run fm
```
